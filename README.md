# 🚀 How to Run

## Clone Repository

```bash
git clone https://github.com/maramsivaramakrishnareddy/brain-tumor-detection.git
cd brain-tumor-detection
```

## Install Dependencies

```bash
pip install -r requirements.txt
```

## Download Model File

Download the pretrained model:

https://drive.google.com/file/d/1LJG_ITCWWtriLC5NPrWxIDwekWbhU_Rj/view

Place the downloaded file inside:

```text
models/
└── bt_resnet50_model.pt
```

## Run Application

```bash
python app.py
```

Open:

```text
http://127.0.0.1:5000
```
